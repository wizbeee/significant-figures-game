QR 코드를 스캔하거나 세션 코드를 입력합니다.

촬영 시작하기

사용 방법

1

노트북에서 세션 시작

노트북 모드를 선택하면 QR 코드가 생성됩니다

2

폰으로 QR 스캔

같은 WiFi에서 폰 카메라로 QR 코드를 스캔합니다

3

촬영 & 분석

얼굴과 전신 사진을 촬영하면 노트북에서 실시간 분석 결과를 확인합니다

폰 카메라 접근은 보안상 HTTPS 또는 localhost에서만 허용됩니다. 로컬 네트워크에서 사용하려면 아래 설정이 필요합니다.

Chrome (Android/PC)

  1. Chrome 주소창에 chrome://flags 입력
  2. #unsafely-treat-insecure-origin-as-secure 검색
  3. 서버 주소 입력 (예: http://192.168.0.10:5000)
  4. Enabled 선택 후 브라우저 재시작

Safari (iPhone/iPad)

설정 → Safari → 고급 → 실험적 기능 → MediaDevices 허용
