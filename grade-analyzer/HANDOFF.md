# 신입생 성적 분석 시스템 — 개발 핸드오프

## 빠른 시작

### 다른 컴퓨터에서 받기
```bash
git clone https://github.com/wizbeee/significant-figures-game.git
cd significant-figures-game
git checkout feat/science-lab-admin-v3.2
```

### 실행
`grade-analyzer/index.html` 파일을 **브라우저에서 직접 열기** (더블클릭)
- 서버 불필요, 인터넷 필요 (CDN 라이브러리)

---

## 현재 구현 상태

### 파일 구조
```
grade-analyzer/
  index.html    ← 단일 HTML 파일 (전체 프로그램)
  HANDOFF.md    ← 이 문서
```

### 기능 목록

| 탭 | 기능 | 상태 |
|---|---|---|
| 📁 데이터 업로드 | 엑셀 드래그앤드롭, 4유형 자동감지, 데모 데이터 | ✅ |
| 📈 모의고사 비교 | 기수별 등급 분포 차트(8종) + 분포표 + 평균 추이 | ✅ |
| 🏷️ 전형별 분석 | 전형별 개요 테이블 + 기수별 차트 | ✅ |
| 📋 내신 분석 | 내신 점수대별 분포 (Type D 데이터 필요) | ⚠️ 기본만 |
| 🏫 출신중학교 | 학교별 인원·1등급 테이블 + 상위15 차트 | ✅ |
| 📝 종합 리포트 | 핵심 발견사항 자동 생성 + 종합 비교표 | ✅ |
| 📤 내보내기 | 엑셀/PPT(차트이미지)/Word/PDF + 미리보기 | ✅ |

### 차트 유형 (8종)
누적 막대(세로/가로), 그룹 막대(세로/가로), 100% 누적, 꺾은선, 방사형, 도넛

### 엑셀 파싱 유형 (4종)
| 유형 | 설명 | 예시 |
|---|---|---|
| Type A | 통합양식, 병합헤더(2~3줄), 73열 | 11기 통합양식(실채점) |
| Type B | RAW파일, 플랫 헤더, 22열 | 12기 예상등급컷 RAW파일 |
| Type C | 복수기수 합본, 병합헤더, 기수 컬럼 | 8,9,10기 학평성적 데이터 |
| Type D | 전형 지원 데이터, 전형구분/출신중/내신 환산점수 | 전형 접수 데이터 |

### 데이터 보안
- 서버 전송 없음 (브라우저 메모리에서만 처리)
- 영구 저장 없음 (localStorage/쿠키 미사용)
- JSON 파일로 수동 저장/불러오기 가능

## CDN 라이브러리
- SheetJS (xlsx) — 엑셀 파싱/내보내기
- Chart.js + chartjs-plugin-datalabels — 차트
- PptxGenJS — PPT 내보내기
- html2canvas + jsPDF — PDF 내보내기

## 남은 작업

### 우선순위 높음
1. **실데이터 파싱 검증** — 실제 11기/12기/합본 파일 업로드 후 수치 대조
2. **Type D 데이터 연동** — 전형 지원 데이터 ↔ 모의고사 데이터 이름 매칭
3. **내신 분석 탭 완성** — Type D의 교과성적/환산점수 기반 차트/테이블

### 우선순위 보통
4. 8기 데이터 파싱 (18시트 엑셀의 "8기" 시트)
5. 차트 렌더링 세부 조정 (색상 가시성, 레이블 겹침)
6. 오프라인 실행 지원 (CDN → 인라인 번들링)
